# Purpose

These scripts launch a specific kubernetes cluster version with an image
registry.

## Deploy KinD with Latest Image

```bash
./kind-with-registry
```

## Deploy Kind with Specific Image

Run any of the provided scripts as such:

```bash
./kind-with-registry-v1.26.sh
```
